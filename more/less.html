# 相同点

Sass和Less都属于CSS预处理器，语法上有很多共性，都支持以下特性，但在具体用法上有些差异

  • 变量
  • 嵌套
  • 混合(mixins)，传参
  • 运算
  • 作用域
  • ...

# 环境差异

# 使用变量

# 变量标识符的差异

@font-size: 20px;

$font-size: 20px;

# 变量插值的差异

$direction: left;
.box {
    padding-#{$direction}: 20px;                             
}

@direction: left;
.box {
    padding-@{direction}: 20px;
}
